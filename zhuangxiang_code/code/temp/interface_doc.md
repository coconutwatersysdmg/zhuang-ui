# 接口处理模式

WCS与机器人系统通过HTTP
POST传递JSON相互通讯,调用地址根据现场实际情况配置；

# 接口调用说明

机器人发送方信息格式：

{

\"id\": \"0\",

\"type\": \"from\",

\"status\": \"empty\"

}

WCS即时返回信息：

{

\"code\": 0,

\"msg\": \"ok\",

\"data\": {}

}

Msg

成功返回ok，失败返回原因

code：

0 : 成功， 非0：失败 ；

data：\
详细错误原因

# 接口分类

  MSGCODE编码        MSG SUB CODE 子编码   描述                                                                         发送方   接收方
  ------------------ --------------------- ---------------------------------------------------------------------------- -------- --------
  库存获取                                 用于XYZ向wcs获取立库内库存信息，箱子信息                                     机器人   WCS
  规划订单输出                             Xyz规划好拼箱信息，纸箱来料顺序，将顺序发送给wcs                             机器人   WCS
  拼箱物料信息下发                         Wcs将机械臂服务器规划好的拼箱信息发送给执行机械臂，机械臂提前规划垛型        WCS      机器人
  物料到达                                 Wcs控制输送线将纸箱送到机械臂抓取位置，发送接口给机械臂机械臂抓取            WCS      机器人
  托盘更新请求                             机械臂向wcs请求更换码垛托盘，或者更新码垛托盘                                机器人   wcs
  托盘到达                                 Wcs控制运输设备将新托盘运输到机械臂码垛位置，Wcs向机械臂发送接口告知机械臂   WCS      机器人
  设备状态                                 机器人反馈当前设备状态，告知wcs，当前是否可以执行任务                        机器人   WCS

### 接口字段

### 接口地址

/adaptor/api/wcs/sendpalletplanresult

### 接口字段

发送

注意:发送数据格式为结构***数组,每个数组元素包含一个case纸箱信息,顺序等,顺序一定是从前往后排列,***

***可能为空 ,空输入为 \[\],代表没有箱子拼case成功***

  字段            类型     是否可为空   默认值       描述
  --------------- -------- ------------ ------------ -------------------------------------------------
  box_index       int      N                         拼case序号(每一批重新更新)
  box_unique_id   string   N                         case唯一识别号
  total_height    real     N                         case拼箱高度
  order_id        string   N                         目视订单号
  case_type       string   N            \"MH423C\"   箱子放置case 型号
  case_group      string   N            0            箱子拼case 组号,如果组号不为0则只能和自己拼case
  layers          list     N                         数组,里面包含case中的每一层
  cartons         list     N                         数组,里面包含case中的每一箱
  length          real     N                         在cartons结构内,箱子长单位mm
  width           real     N                         在cartons结构内,箱子宽单位mm
  height          real     N                         在cartons结构内,箱子高单位mm
  product_code    int      N                         在cartons结构内,箱子产品编号
  layer_id        int      n                         层号
  seq             int      n                         case顺序号

返回:

  字段   字段类型   是否可为空   默认值   描述
  ------ ---------- ------------ -------- -------------------------------
  code   Int        N                     返回状态代码 ,0为正常,非0异常
  msg    string     N                     返回消息事件
